void Police1_task(void *pvParameters) { Event_t event; while(1) { /* Wait for an event from the police queue */ xQueueReceive(xPoliceQueue, &event, portMAX_DELAY); /* Acquire police resource (semaphore) */ xSemaphoreTake(xPoliceSemaphore, portMAX_DELAY); /* Simulate handling the event (1-3 seconds) */ uint32_t duration_ms = 1000 + rand() % 2000; printf("START: Police-1 handling event ID=[%lu]\n", (unsigned long)event.event_id); vTaskDelay(pdMS_TO_TICKS(duration_ms)); printf("COMPLETE: Police-1 finished event ID=[%lu] after %lums\n", (unsigned long)event.event_id, (unsigned long)duration_ms); /* Send event to logger for tracking */ xQueueSend(xLoggerQueue, &event, portMAX_DELAY); /* Release police resource */ xSemaphoreGive(xPoliceSemaphore); } }

Dispatcher Routing Logic

üìÑ Src/tasks.c - Dispatcher Task
void Dispatcher_task(void *pvParameters) { Event_t event; while (1) { /* Receive event from central dispatcher queue */ xQueueReceive(xDispatcherQueue, &event, portMAX_DELAY); /* Route event to appropriate department queue based on type */ switch(event.event_type) { case EVENT_POLICE: xQueueSend(xPoliceQueue, &event, portMAX_DELAY); break; case EVENT_AMBULANCE: xQueueSend(xAmbulanceQueue, &event, portMAX_DELAY); break; case EVENT_FIRE: xQueueSend(xFireQueue, &event, portMAX_DELAY); break; case EVENT_COVID: xQueueSend(xCovidQueue, &event, portMAX_DELAY); break; } } }

üìÖ Implementation Flow

1. Initialization
Create queues, semaphores, and task handles in main()
2. Event Generation
Event generator creates random emergencies with unique IDs
3. Event Routing
Dispatcher routes events to appropriate department queues
4. Resource Acquisition
Department tasks acquire semaphore before handling events
5. Event Processing
Tasks simulate event handling with random delays (1-3s)
6. Logging & Release
Events logged and resources released for next event

üî® Build & Run Instructions

Prerequisites

  • Linux/POSIX environment
  • GCC compiler
  • Make build tool
  • pthread library

Build Steps

  1. Clone the repository:
    git clone https://github.com/Nadavyehoshua/freeRTOS-Emergency-Dispatch.git
  2. Navigate to the project directory:
    cd freeRTOS-Emergency-Dispatch
  3. Build the project:
    make
  4. Run the simulation:
    ./build/posix_demo
  5. Stop with: Ctrl+C
